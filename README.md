# Chat System
A basic WhatsApp-like simple chat System application built with Node.js, Express, MongoDB, and EJS.

## Features
- Create messages
- Edit messages
- Delete messages
- View all chats

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- HTML/CSS

## How to Run
1. Clone the repo
2. `npm install` to install dependencies
3. Start MongoDB
4. Run `node server.js`
5. Open `http://localhost:8080/chats` in your browser
